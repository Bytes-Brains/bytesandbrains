# Code of Conduct

This project adheres to the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants in project spaces (issues, pull requests, discussions, and other community venues) are expected to follow it.

## Enforcement

Report unacceptable behavior to **conduct@bytesandbrains.ai**. All reports will be reviewed and investigated promptly and confidentially.

See the linked Contributor Covenant text above for the full standards, scope, and enforcement guidelines.
